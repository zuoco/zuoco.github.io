IOStream概述

发表于 2025-03-10 更新于 2025-03-30 分类于 cpp ， 流

1.  IOStream

IO可以分为流式I/O, 记录I/O, C++的IOStream采用了流式I/O.

2.   流式I/O的两个问题

2.1.  表示形式

使用“格式化”或“解析”在“数据的内部表示”与“字符序列”之间进行转换。

格式化是将数据的内部表示（二进制表示）转换为字符序列, 解析是将字符序列转换为数据的内部表示，也就是说，cout(输出流)对应的是格式化，cin(输入流)对应解析。

2.2.  与外部设备的通信

所谓外部设备，就是文件、终端、内存等。不同的外部设备，访问的逻辑不同，C++的I/O库进行了封装，对上层提供了统一的接口来访问外部设备。

3.   输入输出涉及的操作

3.1  输出流

格式化 –> 缓存 –> 编码转换 –> 输出

编码转换：UTF-8, GBK.

3.2  输入流

读取 –> 编码转换 –> 解析

4.  IOStream的实现原理
